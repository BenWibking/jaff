## Using JAFF in Your Project

### Attribution

When using JAFF in your project, please include:

1. **The MIT License text** (shown above)
2. **Copyright notice**: "Copyright 2025 Jaff Contributors"

You can include this in:
- Your project's LICENSE file
- A THIRD_PARTY_LICENSES file
- Your software's about/credits section
- Your documentation

### Example Attribution

**In a LICENSE or THIRD_PARTY_LICENSES file:**

```text
This project uses JAFF (Just Another Format):

Copyright 2025 Jaff Contributors
Licensed under the MIT License
https://github.com/tgrassi/jaff
```

**In source code:**

```cpp
/*
 * This code was generated using JAFF
 * Copyright 2025 Jaff Contributors
 * Licensed under the MIT License
 * https://github.com/tgrassi/jaff
 */
```

**In documentation:**

```markdown
## Third-Party Software

This project uses [JAFF](https://github.com/tgrassi/jaff) for
code generation, licensed under the MIT License.
```

## Frequently Asked Questions

### Can I use JAFF in commercial software?

**Yes!** The MIT License explicitly allows commercial use.

### Do I need to open-source my code if I use JAFF?

**No.** The MIT License is permissive and doesn't require you to open-source your code.

### Do I need to credit JAFF in my paper/publication?

While not legally required, we appreciate citations! See [Citation](citation.md) for how to cite JAFF.

### Can I modify JAFF and redistribute it?

**Yes**, as long as you include the MIT License and copyright notice.

### Can I sell software that uses JAFF?

**Yes**, the MIT License allows commercial use and distribution.

### What about the generated code?

Generated code is **your work** and not subject to JAFF's license. You can license it however you want.

### Do I need permission to use JAFF?

**No**, the MIT License grants you permission automatically. Just include the license notice.

### Can I include JAFF in a GPL project?

**Yes**, the MIT License is compatible with GPL and other licenses.
